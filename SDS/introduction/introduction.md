# 1. Introduction

본 문서는 우리 조가 개발하고자 하는 시스템인 Game’s Code System의 design specification(SDS)이다. ‘게임메이트 매칭 시스템’은 웹 기반 서비스로, 설치 없이 브라우저에서 언제든 이용할 수 있는 실시간 파티/듀오 매칭 플랫폼이다. 본 시스템인 ‘게임메이트 매칭 시스템’은 취향과 실력을 기반으로 유저 관점에서 최적의 플레이 파트너를 실시간 및 예약 시스템을 기반으로 연결해 주는 서비스를 목표로 한다. 게임 종류, 성별, 티어, 플레이 시간대 등을 기반으로 유저가 원하는 메이트를 쉽게 찾을 수 있도록 하였다. 또한 유저 본인이 메이트가 되어 다양한 사람들과 매칭 기능을 이용하고 수익을 낼 수 있도록 하였다.
Use case analysis는 사용자 관점에서 소프트웨어가 제공하는 기능을 서술했고 class diagram은 시스템의 구조적 관점, sequence diagram과 state machine diagram은 시스템의 동적 관점을 서술했다. User interface는 사용자 인터페이스의 관점에서 시스템을 서술한다.
