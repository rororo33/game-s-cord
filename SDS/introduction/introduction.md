# 1. Introduction

본 문서는 우리 조가 개발하고자 하는 시스템인 Game’s Code System의 design specification(SDS)이다.

‘게임메이트 매칭 시스템’은 웹 기반 서비스로, 설치 없이 브라우저에서 언제든 이용할 수 있는 실시간 파티/듀오 매칭 플랫폼이다. 본 시스템은 취향과 실력을 기반으로 유저가 원하는 플레이 파트너를 실시간 혹은 예약 방식으로 연결하는 것을 목표로 한다.

게임 종류, 성별, 티어, 플레이 시간대 등 다양한 조건을 고려하여 유저가 원하는 메이트를 쉽게 찾을 수 있도록 하였으며, 유저 본인이 메이트로 등록되어 매칭 기능을 통해 수익을 얻는 것도 가능하다.

Use Case Analysis는 사용자 관점에서 소프트웨어가 제공하는 기능을 서술하고 있으며, Class Diagram은 시스템의 구조적 관점을 나타낸다. 또한 Sequence Diagram과 State Machine Diagram은 시스템의 동적 관점을 설명하고 있다. 마지막으로 User Interface는 사용자 인터페이스의 관점에서 시스템을 설명한다.
