# 8. Gloassary

|      이름      | 설명                                                                                                |
|:------------:|:--------------------------------------------------------------------------------------------------|
| **gamemate** | 이 시스템의 핵심 용어로, 사용자가 특정 게임을 함께 플레이하기 위해 등록하는 ‘게임 파트너’를 의미한다.                                       |
|  **match**   | 사용자가 게임메이트에게 게임 플레이를 요청하는 ‘매칭’ 행위 또는 그 약속 자체를 의미한다. `PENDING`, `ACCEPTED`, `DECLINED` 등의 상태를 가진다. |
|  **payout**  | 매칭이 성사되었을 때, 요청자의 코인이 게임메이트에게 ‘지급’되는 것을 의미한다.                                                     |
| **substate** | 하나의 큰 상태 안에 포함된 작은 단계. 예를 들어 “마이페이지” 안의 “결제내역”처럼 세부 화면이나 단계를 의미한다.                                |
