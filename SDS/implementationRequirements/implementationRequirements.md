# Implementation Requirements

## H/W Platform Requirements

### Development Environment
- **CPU**: 12th Gen Intel(R) Core(TM) i5-12500H or equivalent
- **RAM**: 16GB or more
- **Storage**: SSD 128GB or more

## S/W Platform Requirements

### Development Environment
- **OS**: Windows 10/11+
- **IDE**: IntelliJ IDEA, VS Code

### Technology Stack
- **Backend**
    - **Language**: Java
    - **Framework**: Spring Boot
- **Frontend**
    - **Library**: React
- **Database**
    - **DBMS**: MySQL

## Non-functional Requirements
- Modularity
- Performance
